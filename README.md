# Brownian Aggregation Simulations

Code supporting the PhD work of Sebastian Baudelet.

Implements stochastic coalescence, Brownian aggregation dynamics on a spherical domain and the Cluster-average stochastic coalescence. Includes also plots code for the whole PhD : Modélisation multi-échelles des résaux mycéliens.

### Usage
Core simulation scripts are in `/models`.
